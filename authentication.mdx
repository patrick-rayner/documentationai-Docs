## Overview

Secure your Inkryptus account to protect your crypto assets like INKY tokens, wallet balances, and staking rewards. Inkryptus supports multiple authentication layers including two-factor authentication (`2FA`), wallet recovery phrases, secure logins, and API keys for programmatic access.

<Callout kind="alert">
  Always enable `2FA` immediately after account creation. Never share your recovery phrase or API keys with anyone.
</Callout>

## Authentication Methods

Choose the right method based on your needs. Use the tabs below for platform-specific setup instructions.

<Tabs>
  <Tab title="Two-Factor Authentication" icon="shield">
    Enable `2FA` using an authenticator app like Google Authenticator or Authy.

    <Steps>
      <Step title="Install Authenticator App" icon="download">
        Download and install a trusted authenticator app on your mobile device.
      </Step>
      <Step title="Enable 2FA in Dashboard" icon="settings">
        Log in to your Inkryptus dashboard at `https://dashboard.example.com`.
        
        Navigate to **Account > Security > Two-Factor Authentication**.
        
        Scan the QR code with your app.
      </Step>
      <Step title="Verify Setup" icon="check-circle">
        Enter the `6-digit` code from your app to confirm activation.
      </Step>
    </Steps>
  </Tab>

  <Tab title="Wallet Recovery Phrase" icon="key">
    Your `12-word` recovery phrase backs up your non-custodial wallet. Store it offline securely.

    <Steps>
      <Step title="View Recovery Phrase" icon="eye">
        In the Inkryptus Wallet app, go to **Settings > Security > Recovery Phrase**.
        
        Write down the words in order.
      </Step>
      <Step title="Verify Phrase" icon="check-circle">
        Re-enter the phrase in the app to confirm you have it correct.
      </Step>
      <Step title="Secure Storage" icon="lock">
        Store on paper in a safe location. Never store digitally or share it.
      </Step>
    </Steps>

    <Expandable title="Advanced Recovery Tips" default-open="false">
      Use a metal backup device for fire/water resistance. Test recovery on a new device periodically.
    </Expandable>
  </Tab>

  <Tab title="API Keys" icon="api">
    Generate API keys for automated trading, staking, or wallet integrations.

    <ParamField header="Authorization" param-type="string" required="true">
      Bearer token format: `Bearer YOUR_API_KEY`.
    </ParamField>

    <ParamField query="api_key" param-type="string" required="false">
      Alternative query parameter for legacy endpoints.
    </ParamField>

    Use these keys with the Inkryptus API at `https://api.example.com/v1`.

    <CodeGroup tabs="JavaScript,Python">
      ````javascript
      const response = await fetch('https://api.example.com/v1/wallet/balance', {
        method: 'GET',
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
        }
      });
      const data = await response.json();
      console.log(data);
      ````

      ````python
      import requests

      headers = {
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
      }
      response = requests.get('https://api.example.com/v1/wallet/balance', headers=headers)
      data = response.json()
      print(data)
      ````
    </CodeGroup>
  </Tab>
</Tabs>

## Best Practices for Secure Logins

Follow these guidelines to minimize risks.

<Columns cols={2}>
  <Card title="Use Strong Passwords" icon="lock" href="#">
    Create passwords with at least `12` characters, mixing uppercase, lowercase, numbers, and symbols.
  </Card>
  <Card title="Avoid Phishing" icon="alert-triangle" href="#">
    Always verify URLs start with `https://dashboard.example.com` or `https://inkryptus.com`.
  </Card>
  <Card title="Regular Reviews" icon="settings" href="#">
    Audit connected apps and revoke unused API keys monthly.
  </Card>
  <Card title="Device Security" icon="shield" href="#">
    Enable biometric login and keep your device OS updated.
  </Card>
</Columns>

<Callout kind="tip">
  Monitor login activity in your dashboard under **Account > Recent Activity**. Report suspicious logins immediately.
</Callout>
