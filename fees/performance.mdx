The performance fee is a percentage charged on staking and farming profits. It applies to the daily yield generated by the user's contracts, not to the principal amount.

## Fee Details

| Attribute | Details |
| --- | --- |
| **Amount** | 25% of daily profit |
| **Applies to** | Net yield generated by staking/farming contracts |
| **Deduction** | Automatic, deducted from the daily yield before it is credited |
| **No-profit days** | No fee charged if yield = 0 |

## Example

A user's staking contract generates 10 INKY in profit on a given day:

- Performance fee: 10 INKY x 25% = 2.5 INKY
- User receives: 10 INKY x 75% = 7.5 INKY (net)

## Key Rules

- **No charge on principal**: fee applies only to profit, never to the staked or farmed amount
- **Daily calculation**: fee is calculated independently each day based on that day's yield
- **Conditional**: no fee if the contract does not generate profit on a given day
- **Automatic deduction**: fee is taken before the yield is credited to the user's balance

The performance fee is separate from the Harvest fee (US$3 flat), which is charged when the user collects accumulated rewards. See [Fee Table](fee-table.mdx) for the complete schedule.

For details on how performance fees are distributed to the network, see [Commissions](../partner-program/commissions.mdx).
