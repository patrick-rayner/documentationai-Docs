A flat fee is charged on each transaction operation performed on the Inkryptus platform.

## Fee Details

| Attribute | Details |
| --- | --- |
| **Amount** | US$3 per operation |
| **Currency** | Converted to INKY at market rate at time of execution |
| **Deduction** | Automatic, deducted from the transaction amount |

## Triggering Operations

The transaction fee applies to:

| Operation | Description |
|-----------|-------------|
| **Withdraw** | Moving funds out of the Inkryptus platform |
| **Swap** | Exchanging between assets (BTC, ETH, USDT, INKY, CAKE, etc.) |
| **Harvest** | Collecting staking rewards or farming yields |

Each operation incurs exactly one US$3 fee, regardless of the amount transacted.

## Key Rules

<Callout kind="info">
  The fee is charged at the moment of execution. The US$3 value is converted to INKY at the current market rate for settlement. The fee applies equally to all supported assets.
</Callout>

For details on how transaction fees are distributed to the network, see [Commissions](/partner-program/commissions).
