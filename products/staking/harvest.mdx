Harvest is the process of collecting accumulated staking rewards from an active contract. Only the profit is transferred to the user's free wallet balance. The staked principal remains in the contract.

## How it works

1. User opens the staking dashboard in the Inkryptus app.
2. Accumulated profit is displayed for each active contract.
3. User taps "Harvest" on the desired contract.
4. The fee (US$ 3 flat) is displayed before confirmation.
5. Upon confirmation, the profit is moved to the free balance in the user's Inkryptus wallet.

## Rules

| Rule | Detail |
| ---- | ------ |
| Availability | 24 hours after contract activation |
| Fee | US$ 3 flat per operation |
| What moves | Profit only (principal stays invested) |
| Frequency | Can be performed as often as desired after the initial 24h |
| Compound option | User can manually reinvest harvested profit into new contracts |

## What users can do with harvested profit

- **Reinvest (compound)**: create a new staking contract with the harvested amount.
- **Convert**: use the integrated Buy/Sell flow to swap into other assets (INKY, USDT, BTCB, ETH, CAKE).
- **Withdraw**: transfer to an external wallet (standard withdrawal fee applies).
- **Use in ecosystem**: games, swaps, or other platform products.

## Fee context

The US$ 3 Harvest fee is the same flat fee applied to Swap and Withdraw operations. It covers the on-chain gas cost and platform processing. The fee is always displayed in the app before the user confirms the operation.
