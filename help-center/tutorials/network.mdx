<Callout kind="warning">
  Inkryptus operates exclusively on **BNB Smart Chain (BEP20)**. Transfers on any other network may cause permanent loss of funds.
</Callout>

## Supported assets

All assets are BEP-20 tokens on BNB Smart Chain:

| Asset | Symbol |
|-------|--------|
| Dollar Tether | USDT |
| Bitcoin (BEP-20) | BTCB |
| Ethereum | ETH |
| Inkryptus | INKY |
| PancakeSwap | CAKE |
| Biswap | BSW |
| Radiant | RDNT |
| Shiba Inu | SHIB |

<Callout kind="info">
  This list may be updated over time. Always confirm in the app before transferring.
</Callout>

## Key rules

- Only assets listed above are supported. Unsupported tokens sent to your address may not be credited.
- Always select **BNB Smart Chain (BEP20)** as the network when depositing or withdrawing.
- If unsure about compatibility, send a small test transaction first.
