<ExpandableGroup>
  <Expandable title="How does Coin Flip work?" default-open="true">
    You choose one side (Heads or Tails), define the amount in INKY, review the payout shown on screen, and confirm via **Play**. The result is immediate in the game flow.
  </Expandable>

  <Expandable title="Is there a responsible-use rule?" default-open="false">
    Yes. This feature is intended for users aged 18+. The recommendation is to use only amounts compatible with your profile and strategy.
  </Expandable>
</ExpandableGroup>
