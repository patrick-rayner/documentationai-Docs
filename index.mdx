Welcome to the Inkryptus technical documentation. This is the source of truth for everything technical about the Inkryptus ecosystem: the INKY token, platform products, partner program mechanics, fees and commissions, help content, and legal references.

## Who this is for

This documentation is written for:

- Advanced partners who need operational details
- Investors and researchers evaluating the Inkryptus ecosystem
- Journalists and auditors looking for verifiable facts
- Technical integrators and developers

All commercial narrative lives on the main website (`inkryptus.com`). Educational content for beginners lives in the Learn hub (`inkryptus.com/learn/`). This documentation focuses exclusively on factual, verifiable, and technical information.

## SEO status

This documentation is **not indexed** by search engines. The `robots.txt` blocks all crawlers. This keeps the technical content separate from the commercial SEO strategy and allows for honest, detailed, and occasionally raw technical language without affecting the public brand positioning.

## Conventions

- All numbers that can change are marked as either snapshots (with a date) or as real-time data references (RTD placeholders).
- All addresses are shown in full and linked to BscScan or the relevant explorer.
- All claims about contract behavior are linked to the contract source or to a verified read function on BscScan.
- Updates are logged in the `changelog.md` file of each section.
