## Recent Releases

Stay current with Inkryptus enhancements. Review the latest versions below to see new features, performance boosts, and fixes that improve your crypto management experience.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for multi-chain staking, enabling rewards across Ethereum, Binance Smart Chain, and Polygon.
- Introduced INKY token swap pairs with major assets like BTC, ETH, and USDT directly in the wallet.
- Launched IBO participation dashboard with real-time progress tracking and yield estimates.

## Improvements

- Enhanced wallet security with biometric authentication and hardware wallet integration.
- Optimized staking calculator for faster APY projections using live market data.
- Improved swap interface with slippage protection and gas fee optimization.

## Bug Fixes

- Fixed occasional sync issues in the gaming rewards leaderboard.
- Resolved token balance display errors after cross-chain transfers.
- Corrected transparency report export functionality for CSV and PDF formats.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated push notifications for staking reward claims and swap completions.
- Added portfolio analytics dashboard showing INKY holdings performance over time.
- Enabled team wallet management for collaborative IBO projects.

## Bug Fixes

- Patched vulnerability in swap liquidity pools to prevent front-running exploits.
- Fixed mobile app crashes during high-volume wallet imports.
- Corrected staking reward accrual discrepancies during network congestion.

## Improvements

- Reduced wallet load times by 40% through optimized blockchain query caching.
- Updated UI themes for better dark mode support across all platforms.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Released core wallet with non-custodial storage and seed phrase backup.
- Introduced staking module supporting INKY token locks from 7 days to 5 years.
- Launched initial swap engine with 20+ token pairs.

## Breaking Changes

- Migrated to new API endpoints at `https://api.example.com/v2`; update your integrations.
- Deprecated legacy auth tokens; migrate to `{YOUR_API_KEY}` format.

## Bug Fixes

- Stabilized initial sync for new users.
- Fixed cross-browser compatibility issues in the web app.

</Update>
