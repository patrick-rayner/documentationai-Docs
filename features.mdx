## Overview

Inkryptus provides a suite of powerful features designed for secure and user-friendly cryptocurrency management. You can manage assets in your non-custodial wallet, stake `INKY` tokens to earn rewards, swap tokens effortlessly, and participate in gaming and Initial Blockchain Offerings (`IBO`). These tools empower you to maintain full control over your crypto without unnecessary complexity.

<Callout kind="tip">
  Start with the wallet to securely store your assets, then explore staking and swapping for growth opportunities.
</Callout>

### Staking via API

Integrate staking programmatically:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  import { ethers } from 'ethers';

  const provider = new ethers.JsonRpcProvider('https://api.example.com/rpc');
  const wallet = new ethers.Wallet('YOUR_PRIVATE_KEY', provider);
  const stakingContract = new ethers.Contract('0xStakingAddress', ABI, wallet);

  const tx = await stakingContract.stake('1000000000000000000'); // 1 INKY
  await tx.wait();
  ```
  ```python
  from web3 import Web3

  w3 = Web3(Web3.HTTPProvider('https://api.example.com/rpc'))
  account = w3.eth.account.from_key('YOUR_PRIVATE_KEY')
  staking_contract = w3.eth.contract(address='0xStakingAddress', abi=ABI)

  tx_hash = staking_contract.functions.stake(1000000000000000000).build_transaction({
      'from': account.address,
      'gas': 200000,
      'gasPrice': w3.to_wei('20', 'gwei'),
      'nonce': w3.eth.get_transaction_count(account.address),
  })
  signed_tx = w3.eth.account.sign_transaction(tx_hash, account.key)
  tx_receipt = w3.eth.send_raw_transaction(signed_tx.rawTransaction)
  ```
</CodeGroup>

<Callout kind="success">
  Staked `INKY` tokens contribute to network security and can be unstaked after a `7`-day cooldown.
</Callout>

## Token Swapping

The built-in swap feature aggregates liquidity for optimal rates. Swap `INKY` for other tokens or stablecoins seamlessly.

### Quick Swap Steps

1. Select input and output tokens.
2. Enter amount and review the route.
3. Approve and confirm the transaction.

For developers, use the swap API:

```javascript highlight="3-5" show-lines={true}
const swapResponse = await fetch('https://api.example.com/v1/swap/quote', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
  body: JSON.stringify({
    fromToken: 'INKY',
    toToken: 'USDC',
    amount: '1000000000000000000' // 1 INKY
  })
});
```
