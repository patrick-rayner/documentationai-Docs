## Purpose

INKY is the utility token of the Inkryptus ecosystem. It serves five core functions inside the platform: staking activation, internal fee settlement, game asset, swap pair with USDT on PancakeSwap, and product access gating. Every interaction between the user and the platform's DeFi products passes through INKY in some capacity.
